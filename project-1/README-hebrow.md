## התחלה

### דרישות מוקדמות
* npm
  ```sh
  npm install npm@latest -g
  ```
  
### התקנה
 Clone the repo
```sh
   git clone https://github.com/CHAYA41182/hadasimProject/tree/master/project-1
```
###### צד שרת:
התקנת חבילות נדרשות:
```sh
    cd server
    npm install

 ```

###### צד לקוח:
התקנת חבילות נדרשות:

 ```sh
    cd client
    npm install

 ```
### הפעלה 
###### צד שרת:
```sh
    cd server
    npm start

```
###### צד לקוח:
 ```sh
    cd client
    npm start

 ```
